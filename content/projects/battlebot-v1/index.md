## Overview

BattleBot v1.0 is an academic robotics project developed as part of an integrated engineering project. The goal was to design and build a compact remote-controlled combat robot prototype, combining electronics, embedded programming, mechanical design and system integration.

The project involved the selection and integration of electronic components, actuator control, sensor monitoring, mechanical structure design and practical testing.

## Technical Description

The robot is based on two ESP32-S3 boards. One board is focused on camera streaming and sensor monitoring, while the other manages the main motion control system, the saw mechanism and Bluetooth control modes.

The system includes DC motors, BTS7960 motor drivers, an OV2640 camera module, a Li-Po battery, cooling fans and a custom structure designed for a compact robotic platform.

## Main Features

- Remote-controlled robotic platform.
- Dual ESP32-S3 architecture.
- DC motor control using high-current motor drivers.
- Camera module for video streaming.
- Sensor and actuator integration.
- Custom mechanical structure with 3D-printed parts.
- Compact combat robot design.

## What I Learned

This project allowed me to improve my understanding of embedded systems, actuator control, component integration and practical troubleshooting. It also helped me connect different engineering areas: electronics, programming, mechanical design and control.

One of the most valuable parts of the project was learning how to move from an initial concept to a working prototype, dealing with real hardware limitations, wiring, power supply requirements and mechanical constraints.
