## Overview

This project consisted of developing a Pacman-style game from scratch using VHDL on an FPGA platform. The game was displayed on a VGA monitor and required the design of several digital logic blocks.

The main goal was to understand how digital systems can be designed at hardware level, working directly with timing, synchronization, state machines and graphic generation.

## Technical Description

The project included the implementation of VGA synchronization signals, graphic elements, game logic, player movement and basic game states.

Unlike software programming, this project required thinking in terms of hardware behavior, parallel logic and clock-based design. Each part of the system had to be designed considering timing constraints and the way signals evolve in digital circuits.

## Main Features

- Pacman-style game developed from scratch.
- VHDL-based digital design.
- VGA video output.
- Graphic generation using digital logic.
- Player movement and game states.
- Finite state machine design.
- Timing and synchronization control.

## What I Learned

This project helped me understand the difference between software programming and hardware description. It improved my ability to design sequential and combinational logic, structure a digital system and manage timing constraints.

It was especially useful for learning how complex behavior can be built from basic digital blocks when working with FPGA-based systems.
